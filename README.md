# The Bullie Brigade: Sky Whale Relic Run

A complete interactive web artwork/game made for a final project. It uses HTML, CSS, JavaScript, Canvas, JSON data, classes, arrays, animation, keyboard input, mouse interaction, scoring, collision detection, and multiple stages.

## How to Run

Open `index.html` in a browser, or right-click `index.html` in VS Code and choose **Open with Live Server**.

## Controls

- Move with Arrow Keys or WASD
- Collect glowing relic shards
- Avoid purple Odium shadow hazards
- Click on the canvas to release a bubble-pipe pulse
- Reach 120 relic power before morale runs out

## Course Concepts Used

- JavaScript logic and control flow
- Classes and objects
- Arrays and iteration
- Keyboard and mouse input
- Canvas drawing and animation loop
- Collision detection
- JSON-loaded data
- Scoring and game states
- Multiple stages / escalating difficulty
- Visual feedback and optional sound effects

## Student Process Summary

I created this project as an interactive artwork inspired by *The Bullie Brigade*. I used Canvas to draw the Sky Whale scene and JavaScript to control movement, collisions, scoring, collectible relics, hazards, and changing mission states. JSON files are used to load relic and hazard data so the project includes external data instead of hard-coding every detail directly into the script.
