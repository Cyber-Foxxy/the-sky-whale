The Bullie Brigade: Sky Whale Relic Run

Help Moose and Zephyr stabilize the Sky Whale by collecting ancient relic energy, dodging Odium shadow hazards, and keeping the ship's power balanced.

Mission Controls

Move: Arrow keys or WASD

Collect: glowing relic shards

Avoid: purple Odium shadows

Click: release a bubble-pipe pulse

Goal: reach 120 relic power before morale runs out.

Project Concept

This interactive artwork combines canvas animation, JavaScript classes, arrays, JSON-loaded objects, keyboard and mouse input, collision detection, scoring, multiple stages, and visual feedback. The scene is inspired by The Bullie Brigade: a steampunk adventure aboard the Sky Whale.
